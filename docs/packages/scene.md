# @wity/scene

Convenience re-export of the full `@wity/scene-core` public API.

```bash
npm install @wity/scene
```

```js
import { parse, serialize, evaluate, validate } from '@wity/scene';
```

All exports are identical to `@wity/scene-core`. Use this package when you want a single clean import and do not need to reference internal sub-paths.
